# Find the Vowels

A function that returns the number of vowels used in a string.
<br />Vowels are the characters 'a', 'e', 'i', 'o', and 'u'.

##### --- Example

`vowels('Hi There!')` --> 3<br />
`vowels('Why do you ask?')` --> 4<br />
`vowels('Why?')` --> 0
