# Steps

A function that accepts a positive number N and console log a step shape with N levels using the # character.

##### --- Example

`steps(2)`<br />
'#&nbsp;&nbsp;'<br />
'##'<br /><br />
`steps(3)`<br />
'#&nbsp;&nbsp;&nbsp;&nbsp;'<br />
'##&nbsp;&nbsp;'<br />
'###'<br /><br />
`steps(4)`<br />
'#&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;'<br />
'##&nbsp;&nbsp;&nbsp;&nbsp;'<br />
'###&nbsp;&nbsp;'<br />
'####'<br />
